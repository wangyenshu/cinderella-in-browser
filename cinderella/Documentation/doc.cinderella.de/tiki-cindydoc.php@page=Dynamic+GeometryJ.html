シンデレラによるインタラクティブな幾何学


　シンデレラの最も基本的で主要な使い方は、やはりインタラクティブな幾何学です。それは動的幾何学とも呼ばれています。わずかなマウスクリックで、基本的な幾何学から射影幾何学、さらには双曲幾何学やフラクタルも構成できます。シンデレラの主要な特徴は「動かすモード」です。ドラッグモードとも呼ばれます。このモードは、普通の幾何のスケッチツールより多くのものを与えます。図形の基本要素をドラッグすると、関連する要素が動きます。このモードを用いれば、多くの幾何学的構造を探索し実験的なレベルに対する影響と定理を観察することができます。
　以下の説明は、インタラクティブな幾何学に利用できるすべてのツールとその可能性について詳しい案内となります。基本的なモードの説明の後、図形のいろいろな見方についての説明があります。
　Cinderella.2 で新しくなったこととして、幾何変換があります。これを用いて、変換群やフラクタル図形を簡単に作ることができます。
