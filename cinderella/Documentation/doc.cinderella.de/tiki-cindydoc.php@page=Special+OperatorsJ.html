特殊演算子


この節では、これまでの分類に入らない演算子を紹介します。それでも、それらはとても有用で、いくつかはシンデレラと CindyScript の相互関係のために必要です。


  • Interaction with GeometryJ: Cinderella（幾何）と CindyScript の相互関係
  • File ManagementJ 　　　　 : ファイルの読み書き
  • Console OutputJ　　　　　 : CindyScript のコンソールへの出力
  • Timing and AnimationsJ 　 : 内部時計へのアクセス
  • User inputJ　　　　　　　　: キーボードとマウスへのアクセス
  • System Information　　　　:システム情報
  • Interaction with CindyLabJ : CindyLab と CindyScript の相互関係
