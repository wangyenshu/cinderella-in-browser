Arc


This mode is very similar to circle by three points mode, except that here, the three points are not used to define a circle. They are used to define an arc.

Three clicks are necessary to define the arc. The first and the last click define the start and end of the arc. The second click defienes a point that should lie between the other two on the arc.


Synopsis


An arc by three clicks that select three points.
