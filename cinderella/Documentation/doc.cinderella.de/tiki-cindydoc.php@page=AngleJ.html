角度


このモードでは2直線がはさむ角度を計算します。 このモードを使うのには多少の注意が必要です。 2直線がはさむ角というのは一般に2つありうるからです (互いに補角の関係にあります)。 シンデレラはこのことを考えに入れているので、角度モードで知りたい角度を指定する方法は、 角の2等分線の時と同じく選択した位置に依存しています。 つまり、測る角度を指定するには2直線を選択する必要があるのですが、 選択するときにクリックした2点と2直線の交点のなす三角形の内角が指定された角になります。 直感的な操作で十分です。

シンデレラでは画面上にヒントを出して測る角度を選びやすくしています。

最初の選択:
直線がハイライトします
選択した点が記憶されます
 マウスを動かす:
どの角度が指定されるか
ヒントが現れます 		第2の選択:
角が表示されます



要約

二直線のなす角を求める

注意

選択する幾何学によって角度の定義が異なるので注意が必要です。 また、双曲幾何学の場合には角度が複素数で表示されることもあります。
