インストールとアップデート


シンデレラはJava言語で書かれています。大部分のコンピュータではJAVAを走らせるための環境が事前にインストールされています。 これは Mac OS X とほとんどの Windowsを含んでいます。その他のオペレーティングシステム(たとえば Linux) では、無料の Java 仮想マシンを http://www.java.com からダウンロードしてインストールしなければならないかもしれません。

シンデレラソフトウェアそのものはCinderella homepageホームページから無料でダウンロードできます。あなたが使っているコンピュータ(OS)に応じて、インストーラかアプリケーションそのものをダウンロードしてください。

ライセンスの取得

　最初にダウンロードしたシンデレラは、標準的な幾何学モードだけが動作します。物理シミュレーションとプログラミング言語CindyScriptは実行できません。また、いくつかの幾何学モードは使用できません。この制限をはずすためには、正当に取得したライセンスコードを入力する必要があります。ライセンスコードはシンデレラのホームページの at "Buy a License"で取得できます。
　ライセンスは簡単なCinderellaのファイルとして送付されます。これを一度開けば自動的に認証が行われます。ライセンスは３年おきに更新されます。（再購入の必要はありません）　ライセンス情報は、コンピュータの内部に保存され、シンデレラの新しい版にアップデートするときもそのまま使うことができます。


ネットワークライセンス

ネットワークの環境で、一人一人がライセンスコードを入力するのは不便です。そこで、 cinderella2-global.propertiesというファイルにライセンスキーをコピーすることにより、全体で使えるようになります。このファイルは、 cinderella2.propertiesやソフトウェアの初期設定と同じフォルダになければなりません。 通常はこれはインストールしたフォルダにあります。 Mac-OSXでは、ファインダのシンデレラのアイコンをCtrlキーを押しながらクリックすると、ウィンドウが出ますので、 "パッケージの内容を表示" を選ぶと、Contents/Resourcesフォルダがあります。

シンデレラのアップデート

私たちは、新機能の追加やバグ・フィックスのために、Cinderella.2を連続的に更新しています。できるだけ頻繁にダウンロードページ Download Cinderella.2 ページをチェックして、常に最新版を使うようにしてください。
