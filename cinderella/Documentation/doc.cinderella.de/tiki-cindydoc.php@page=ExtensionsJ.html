拡張機能


　Cinderellaの機能を拡張するいくつかの方法があります。プラグインを書いたり、Javascript と連携するなど、他のスクリプト言語が使えるでしょう。

　この節では、これらの可能性について簡単に述べます。これらの機能は、今後まだまだ変わっていくでしょう。ですから、最新の情報を得るためにオンラインマニュアルを参照することを勧めます。


  • プラグイン (Cinderellaのプラグイン技法)
  • Javascript (ブラウザとの連携)
  • インタラクティブな練習問題 (学生用の練習問題の作り方)
  • マルチタッチサポート (Cinderellaでマルチタッチを使う)
  • MatheVital (Cinderellaを使った多数の例)
  • Jython and JRuby (他のスクリプト言語)
  • Visage (グラフアルゴリズムの視覚化)
  • CleverPHL Framework のサポート (録音と編集、再生)
