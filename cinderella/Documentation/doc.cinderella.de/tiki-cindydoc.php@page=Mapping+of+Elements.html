Mapping of Elements


As already explained in the section General Use of Transformations, it is easy to map an element in move mode. Simply select it and press the desired view button of the transformation.

There is also a special mode, map an element, that can be used to map an element. In this mode one has to select an element and the transformation. This selection results in the generation of the mapped element.
