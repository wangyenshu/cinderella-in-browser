Circle by Fixed Radius


This mode allows a circle that has a fixed, numerically given, radius to be created. The new circle and its center are added with a single press–drag–release sequence.

When you select this mode, the property inspector pops up, asking you to specify the desired radius, as in add a line with a fixed angle mode. After that the use of this mode is analogous to that of add a point mode.


Synopsis


Circle by fixed radius mode creates a circle with numerically given fixed radius with a press–drag–release sequence.



Caution


The "shape" of the circle depends on the choice of the geometry.
