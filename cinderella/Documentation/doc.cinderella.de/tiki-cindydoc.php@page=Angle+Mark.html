Angle Mark


This mode creates a graphical mark for an angle. The mark itself is not geometrically active and cannot be used as input for other constructions.

Three clicks at three points are necessary to define the angle mark. The middle point marks the one at which the angle mark is drawn. The other two points define the start and the end of the angle mark.


Synopsis


An angle mark by three clicks that select three points.
