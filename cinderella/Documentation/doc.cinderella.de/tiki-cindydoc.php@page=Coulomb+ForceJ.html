クーロン力


クーロン力は荷電粒子間の力をモデル化します。 CindyLabでは、クーロン力は、特殊な種類の バネとしてモデル化された相互作用です。この力は、粒子の電荷とそれらの距離に依存します。電荷が同符号なら反発し、異符号なら引きあいます。力の絶対値は次の公式で計算されます。


力 = バネ定数 * 電荷1 * 電荷2 / 距離²




詳細は バネの項目を参照してください。
