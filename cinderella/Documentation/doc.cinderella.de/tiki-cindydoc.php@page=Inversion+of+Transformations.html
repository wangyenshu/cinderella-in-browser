Inversion of Transformations


The mode inverse enables one to construct the inverse of any transformation. For this one has simply to choose this mode and click on the transformation to be inverted. Thus a new transformation that is the inverse of the selected one is generated. This operation is frequently useful in studying the group structure underlying a set of transformations.
