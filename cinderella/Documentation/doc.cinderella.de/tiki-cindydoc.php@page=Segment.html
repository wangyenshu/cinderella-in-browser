Segment


This mode is completely analogous to add a line mode. By a press–drag–release sequence you add two points and a segment joining them. In addition, segments can also be supplied with arrows at one or at both ends. The arrow type can be chosen in the inspector tab for special appearance.

In contrast to the old release of Cinderella, segments are now fully active geometric elements. They can be used like any other geometric object. Intersections with segments disappear when the segment is not hit by the intersecting object, as do all elements depending on these intersections. This creates the possibility of allowing geometric objects to appear and disappear depending on certain geometric relations. However, it is much more convenient to create these kinds of optical effects using CindyScript.
