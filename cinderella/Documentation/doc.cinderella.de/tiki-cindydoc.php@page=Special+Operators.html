Special Operators


This section presents a collection of operators that do not fit into any of the other categories. Nevertheless, most of them are extremely useful. Some of them are even necessary for the interaction of Cinderella and CindyScript.


  • Interaction with Geometry: How Cinderella and CindyScript can influence each other
  • File Management: Input and output from and to files
  • Console Output: Printing messages on the CindyScript console
  • Timing and Animations: Handling of timed events
  • User Input: Keyboard and mouse handling
  • System Information: Obtaining information about the runtime system
  • Interaction with CindyLab: How CindyLab and CindyScript can influence each other
