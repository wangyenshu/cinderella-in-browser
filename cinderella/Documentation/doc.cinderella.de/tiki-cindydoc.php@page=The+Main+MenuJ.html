メインメニュー


シンデレラのすべての機能は、画面上部のメイン・メニューによってアクセスできます。シンデレラでこのマニュアルを読んでいる方はすでにメニューの使い方はご存知でしょう。この節では、メインメニューの内容について簡単に説明します。

メインメニュー



ファイル

ファイルメニューには、作図したものの保存／読み込み、画像の読み込みなどのメニューがあります。
ファイルメニューからは、作図したものをいくつかの画像形式で保存したり、印刷したり、ＨＴＭＬページを作成したりできます。
ファイルメニュー



編集

このメニューには、作図の複写や貼り付けのメニューがあります。さらに、点や線の選択操作があります。 インスペクタウィンドウもここから開けます。
編集メニュー



モード

モードメニューには、作図のためのすべての道具がそろっています。幾何の作図の他、物理シミュレーション CindyLab のための道具もあります。これらの道具の多くは、 ツールバーにも用意されていますが、多角形や変換など、ツールバーにはないものもあります。それぞれサブメニューがあります。
モードメニュー


表示

ここでは、図を表示する異なる画面を開くことができます。ユークリッド表示のほか非ユークリッド表示、幾何要素の一覧表、自動証明機能の情報などです。詳しくは 画面表示の節を参照してください。
表示メニュー


スクリプト

CindyScript など、シンデレラで使えるいくつかのスクリプト、カスタムツールの保存／読み込みメニューがあります。グラフアルゴリズムのVisageへはここから行けます。
スクリプトメニュー


設定

このメニューで、長さや角度の単位、方程式の書式などを選んだり、ユークリッド幾何や双曲幾何、楕円幾何など（初期値はユークリッド）を選びます。それらの性質については、 幾何学 の節をお読みください。また、ツールバーをカスタマイズするメニューもあります。

設定メニュー


ヘルプ

選択しているツールの説明、マニュアルの参照、Cinderellaのサイトへの入口があります。
ヘルプ
