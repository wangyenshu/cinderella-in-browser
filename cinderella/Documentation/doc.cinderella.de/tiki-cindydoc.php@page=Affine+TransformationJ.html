アフィン変換


アフィン変換は 相似変換 と 射影変換の中間的位置にあります。 相似変換は２直線のなす角を保ちますが、アフィン変換ではそうではありません。しかしながら、平行は、射影変換の場合ではなく、アフィン変換によって保たれます。

アフィン変換は点の前の像と現在の像の３つの組によって定義されます。 その定義は相似変換と類似しています。(相似変換は４つ、アフィン変換は６つの点を必要としますが)
