線分を加える


このモードは add a lineJ モードとほとんど同じです。「押す、ドラッグする、離す」という一連の作業で2つの点とそれを結ぶ線分をを新しく描きます。さらに、線分には片方または両方の端点に矢印をつけることができます。矢印の形は、インスペクタの特別な表示の設定で選択できます。

シンデレラの旧版と異なり、線分は完全にアクティブな要素となっており、他の要素と同じように扱えます。線分と交差する図形がなければ交点はできません。これは正確な幾何学的関係によって、オブジェクトを現れたり消えたりさせる可能性を生みます。しかし、CindyScript を使ってこのような光学的特殊効果を作る方がずっと便利です。
