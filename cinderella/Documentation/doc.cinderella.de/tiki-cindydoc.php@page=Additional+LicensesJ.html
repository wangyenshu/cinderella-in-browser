Cinderellaで使われているツール


　次のツールがCinderellaの開発で使われました。 Cinderella は 1996年以降、 Java で書かれています。 ( http://java.sun.comをごらん下さい) 以来、15年の間それに満足しています。かなり長い期間になります。初期の版はObjective-Cで書かれていました。そこに戻ることもありえます。

2003から、 Mac OS X 上で開発をしています。( http://www.apple.com/macosxを参照)　それ以前はLinuxでした。

IntelliJ IDEA ( http://www.jetbrains.com参照)は、優れたJava の開発環境です。エディタは Emacs。プログラムを書くのに、そのリファクタリンツ・ツ−ルに依存しています。それがなければ Cinderella.2 はできていませんでした。

ProGuard version 3.5 ( http://proguard.sourceforge.net参照) は used to shrink and obfuscate the Java byte code of Cinderella. This ensures much shorter load times for the Applet, better performance for the standalone version, and a little bit of IP protection for us.

Our builds are generated with Apache Ant (see http://ant.apache.org). Our bugtracking is done using a courier imap server. install4j is used to provide an installer for Windows and Linux. On Mac OS X, there is no installer necessary.

TikiWiki is used not only for the Cinderella web site, but also for preparing the online documentation. We are using TikiWikiToLatex by Stephan Berndts to convert the TikiWiki database into LaTeX and then convert this using pdflatex into the camera-ready manuscript.
