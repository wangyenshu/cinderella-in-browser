Polar of a Line


This mode constructs the polar of a line with respect to a conic. The line and the conic can be selected in any order. This mode can be used to construct the point where a tangent line touches the conic.


Synopsis


The polar point of a line with respect to a conic is constructed by selecting a line and a conic.
